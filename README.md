# Django To-Do App

This is a simple To-Do web application built using Django.

## Features
- Add tasks
- View tasks
- Mark tasks as completed
- Delete tasks

## Technologies Used
- Python
- Django
- HTML
- CSS
- SQLite

## How to Run

1. Install Django
   pip install django
2. Run migrations
   python manage.py migrate
3. start server
   python manage.py runserver
4. open in browser
   http://127.0.0.1:8000/

## About
This project was created as a beginner Django practice project to understand CRUD operations.
